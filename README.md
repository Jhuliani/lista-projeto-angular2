# Projeto Angular II - Ada Tech - Prof. Ivirson

Lista de compras desenvolvida em Angular, utilizando Json-Server como backend.

Para rodar a aplicação, é necessário ter instalado o Json Server:
rodar no terminal:

<code>npm i json-server </code>

Inicializar a aplicação do backend e do front com:

<code> npm start </code>
